jquery.auto_html.js
===================

jQuery port of Rails' extension [auto_html](http://github.com/dejan/auto_html)

## Example usage

    <script type="text/javascript">
      $('.comment')
        .simpleFormat()
        .image()
        .youtube({width:500, height:400})
        .link();
    </script>
